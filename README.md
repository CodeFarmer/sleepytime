# sleepytime

Simple command line sleep tracker

## Installation

Download from http://example.com/FIXME.

## Usage

    $ java -jar sleepytime-0.1.0-standalone.jar 21:30 06:00
    $ java -jar sleepytime-0.1.0-standalone.jar 21:30 06:00 false "Beatrix woke me up"
